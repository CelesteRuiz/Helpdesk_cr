Sistema Help Desk - Gestión de Incidentes

Este proyecto consiste en el diseño de un sistema de gestión de incidentes para un Data Center.

Funcionalidades
- Creación de tickets
- Asignación de técnicos
- Gestión de estados

Patrones de diseño
- Singleton
- Factory Method
- Observer

 Metodología
Se utiliza Gitflow para el control de versiones.
